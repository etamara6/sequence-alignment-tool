# 🧬 Sequence Alignment Tool

A genomic sequence alignment tool implementing three algorithms from scratch:

- **Needleman-Wunsch** — global alignment
- **Smith-Waterman** — local alignment  
- **BLAST-like heuristic** — fast approximate search

## Files
- `sequence_alignment.py` — Python backend with NumPy optimization
- `App.js` — Interactive React visualizer

## How to run
**Python:** `python sequence_alignment.py`  
**Visualizer:** `npm install` then `npm start`

## Built with
Python · NumPy · React
